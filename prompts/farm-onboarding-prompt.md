# Farm onboarding prompt

```text
Help me configure AgriAgentMesh for my farm.

Ask me for the minimum information needed to create:
1. farm_profile.yaml,
2. device inventory,
3. agent list,
4. safety policy,
5. cloud and edge deployment plan,
6. AI provider configuration,
7. first three automation workflows.

Do not assume I have all hardware installed. Provide three options: sensor-only pilot, UAV monitoring pilot, and full cloud-edge-field automation.

For every physical action, mark it as recommendation-only until I explicitly enable approval and execution.
```
